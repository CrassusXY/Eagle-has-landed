\doxysubsubsection{\texorpdfstring{tick()}{tick()}}
{\footnotesize\ttfamily void lander\+::tick (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Updates the state of the lander. 

This function updates the state of the lander based on its current position, velocity, rotation, engine status, and fuel level. It adjusts the velocity in the X and Y directions based on the engine status and applies gravity when the engine is off or fuel is depleted. The lander\textquotesingle{}s position is updated accordingly, and if it lands successfully, the score is set, the landed status is updated, and the lander\textquotesingle{}s pixmap is set to the default lander image.
